# Vesuvius Schul-Demo

Statische Browser-Demo fuer eine Unterrichtseinheit zur Vesuvius Challenge.

## GitHub Pages

Die Seite kann direkt aus dem Repository-Root ausgeliefert werden:

- Einstieg: `index.html`
- Keine Build-Schritte noetig
- `.nojekyll` ist gesetzt, damit GitHub Pages die Dateien unveraendert ausliefert

Station 1, Beispiel 3 nutzt einen stark verkleinerten echten Ausschnitt aus offiziellen Vesuvius-Challenge-Daten. Die anderen Stationen verwenden synthetische Dummy-Daten mit bekannter Wahrheit.
